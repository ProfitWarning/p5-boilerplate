# Processing p5.js Boilerplate

Boilerplate project for [p5.js](https://p5js.org/).

## Development server

```
$ npm install
$ npm start
```

Now you can go to `http://localhost:8080`.

## Build

```
$ npm install
$ npm run build
```

Build target folder is `build`.
